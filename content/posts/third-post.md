## 课程收获
通过本次实践作业，我初步掌握了以下技能：
- GitHub 仓库的创建与管理
- Markdown 文档的编写
- GitHub Actions 工作流的配置
- GitHub Pages 静态网站托管

## 遇到的问题
最初在使用 git clone 下载主题时，不小心把主题文件夹变成了子模块，导致 GitHub Actions 构建失败。后来改为直接下载 ZIP 文件解压，问题得以解决。

## 总结
开源工具非常强大，只要肯动手实践，就能搭建出属于自己的个人网站。
